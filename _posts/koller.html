1 Declarative representation

It is a model of the system, where the model encodes knowledge and reasoning.

1.1 Task

Reason probabilistically about the values of a variable given observation about some other. Example: what is the probability that \(X\) will take a value \(x\) given that you have some information over another variable \(Y\) (posterior distribution).
